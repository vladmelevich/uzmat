# Инструкция по запуску через Docker

## Быстрый старт

1. Убедитесь, что Docker и Docker Compose установлены на сервере

2. Клонируйте проект или загрузите файлы на сервер

3. Перейдите в папку проекта:
```bash
cd /var/www/uzmat
```

4. Запустите контейнеры:
```bash
docker-compose up -d --build
```

5. Сайт будет доступен по адресу: `http://109.199.127.149`

6. Админка доступна по адресу: `http://109.199.127.149/admin/`
   - Логин: `admin`
   - Пароль: `12133`

## Полезные команды

Остановить контейнеры:
```bash
docker-compose down
```

Посмотреть логи:
```bash
docker-compose logs -f
```

Перезапустить контейнеры:
```bash
docker-compose restart
```

Выполнить команду в контейнере:
```bash
docker-compose exec web python manage.py <команда>
```

## Структура

- `Dockerfile` - образ для Django приложения
- `docker-compose.yml` - конфигурация для запуска всех сервисов
- `nginx.conf` - конфигурация Nginx
- `entrypoint.sh` - скрипт запуска (миграции, статика, суперпользователь)
- `create_superuser.py` - скрипт создания суперпользователя

## Автоматически выполняется при запуске

1. Применение миграций базы данных
2. Сборка статических файлов
3. Создание суперпользователя (admin/12133)

# Uzmat - Премиальный маркетплейс спецтехники

Премиальный сайт для объявлений по спецтехнике с премиальным дизайном в зелёно-чёрно-белых тонах.

## Быстрый старт

### 1. Установка зависимостей (один раз):
```bash
pip install Django djangorestframework
```

### 2. Применение миграций (один раз):
```bash
python manage.py migrate
```

### 3. Запуск сервера:
```bash
python manage.py runserver
```

Сайт будет доступен по адресу: **http://127.0.0.1:8000/**

**Всё настроено и готово к работе!**

## Структура проекта

- `uzmat_site/` - главный проект Django
- `uzmat/` - приложение с объявлениями
- `templates/uzmat/` - HTML шаблоны
- `static/uzmat/` - CSS и JavaScript файлы

## Особенности

- Премиальный дизайн в зелёно-чёрно-белых тонах
- Полная адаптивность (мобильные, планшеты, десктоп)
- Плавные анимации и ощущение мобильного приложения
- Готово для интеграции с Django REST Framework

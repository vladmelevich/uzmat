# Папка для скриншотов

Поместите сюда скриншоты страниц сайта для презентации.

## Необходимые скриншоты:

1. **main_page.png** или **main_page.jpg** - Главная страница (для слайда "О проекте")
2. **index_full.png** или **index_full.jpg** - Полный скриншот главной страницы
3. **catalog.png** или **catalog.jpg** - Страница каталога
4. **parts.png** или **parts.jpg** - Страница запчастей
5. **repair.png** или **repair.jpg** - Страница ремонта/услуг
6. **profile.png** или **profile.jpg** - Страница профиля пользователя
7. **create_ad.png** или **create_ad.jpg** - Форма создания объявления
8. **ad_detail.png** или **ad_detail.jpg** - Детальная страница объявления
9. **chats.png** или **chats.jpg** - Страница чатов/заявок

## Формат:
- Поддерживаются форматы: PNG, JPG, JPEG
- Рекомендуемое разрешение: 1920x1080 или выше
- Скриншоты должны быть четкими и хорошо читаемыми

## Как сделать скриншоты:

1. Откройте сайт в браузере
2. Перейдите на нужную страницу
3. Сделайте скриншот (F12 для DevTools, затем Ctrl+Shift+P → "Capture screenshot")
4. Сохраните файл с соответствующим именем в эту папку
5. Запустите скрипт `create_presentation_with_images.py`

Если скриншоты не будут найдены, презентация все равно создастся, но с текстовыми описаниями вместо изображений.
